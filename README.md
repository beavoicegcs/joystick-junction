## 🛠️ Features
- **Zero Badges:** Unlike Wix or Framer, this site is 100% ad-free.
- **Neon Gaming Aesthetic:** A dark-themed UI with high-contrast neon accents.
- **Fully Responsive:** Designed to look great on PC, Tablet, and Mobile.
- **Fast Load Times:** No heavy third-party scripts, just pure HTML and CSS.

## 📂 Project Structure
* `index.html` - The main structure of the website.
* `style.css` - All the custom styling, colors, and layout rules.
* `/assets` - (Optional) Folder for game screenshots and logos.

## 🔧 How to Customize
1. **Change the Colors:** Open `style.css` and look for the `#00ffcc` hex code to swap the neon green for your own signature color.
2. **Update Content:** Edit the text inside the `<section>` tags in `index.html`.
3. **Add Your Socials:** Replace the `#` in the `<a>` tags with your actual Discord or Twitch links.
